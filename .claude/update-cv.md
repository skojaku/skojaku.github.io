# Update the cv in my homepage

## Steps to update the CV on my homepage 
- Compile the latest version of my CV (./libs/skojaku-cv/main.tex) using LaTex
- If the latex file for the CV is updated, git add the changes to the skojaku-cv repository 
- Move the compiled PDF file (./libs/skojaku-cv/main.pdf) to the static files directory of my homepage (./static/files/skojaku-cv.pdf)
- Commit the changes to my homepage repository with a message indicating the CV has been updated and push to the remote repository
